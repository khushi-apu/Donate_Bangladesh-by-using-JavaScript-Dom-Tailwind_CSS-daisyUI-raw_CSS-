What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.With the Document Object Model, programmers can create and build documents, navigate their structure, and add, modify, or delete elements and content. Anything found in an HTML or XML document can be accessed, changed, deleted, or added using the Document Object Model, with a few exceptions - in particular, the DOM interfaces for the internal subset and external subset have not yet been specified.

How do you select an element from the DOM?

Selecting an element from the DOM involves using methods provided by JavaScript to target and retrieve elements based on specific criteria such as ID, class, tag name, or CSS selectors. Depending on the selection method used, you can either retrieve a single element or multiple elements that match the criteria. Some methods return a live collection of elements that updates if the DOM changes, while others return a static collection. By using these methods, you can access elements and manipulate their properties, attributes, styles, or content, enabling dynamic interaction and updates to the web page.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of adding individual event listeners to each child element, the parent element listens for events that bubble up from its children. This is possible because of event propagation, specifically event bubbling, where an event triggered on a child element bubbles up through the DOM hierarchy, reaching its ancestors. Event delegation is useful because it improves performance by reducing the number of event listeners, especially when handling a large number of child elements.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element's attributes and styles in the DOM involves interacting with an element's properties to dynamically change its behavior and appearance. For attributes, JavaScript allows you to retrieve, set, or remove attributes from elements. By modifying an element’s attributes, you can adjust properties such as src for images, href for links, or custom data-* attributes used for storing extra information. For styles, you can modify an element's inline styles directly using the style property, allowing you to set or change specific CSS properties like color, font size, or margins.
